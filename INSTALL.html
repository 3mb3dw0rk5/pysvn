Building pysvn Extension

Prerequisites

To build pysvn you will require:

  • PyCXX V5.2
  • rapidsvn current for the svncpp library
  • rapidsvn subversion 0.34.0

Building on win32

  1. build subversion 0.34
  2. checkout the latest rapidsvn sources
  3. build rapidsvn
  4. fetch/expand the pysvn source code into extdir
  5. Expand pycxx_5_2_0.tar.gz into extdir\Import
  6. edit Builder\builder_custom_init.cmd to match the locations of the sources.
    Note: you must build with MSVC6 as python 2.3 was built with that version.
    MSVC 2003.NET is not expected to work until there is a version of python built VC++ .NET released (issues of C runtime compatibility).
  7. cd Builder
  8. run builder_custom_init.cmd
  9. run nmake build

Building on linux systems. I have built and tested on Redhat 9 with Python 2.3.

  1. build subversion 0.34
  2. checkout the latest rapidsvn sources
  3. build rapidsvn
  4. tar xzf pycxx_5_2_0.tar.gz to extract the PyCXX source code for pycxx
  5. get the pysvn source code
  6. edit pysvn_linux_py23.mak match the locations of the sources and libraries.
  7. make -f pysvn_linux_py23.mak all
